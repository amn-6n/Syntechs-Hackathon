# LearnSphere - AI-Powered Gamified Quiz Learning Platform

**Created by**: Aman Singh

## 📚 What is LearnSphere?

LearnSphere is an intelligent, gamified learning platform designed to help students **practice and master MCQ-based exams** through an engaging, game-like experience. It combines **AI-powered quiz generation** with **interactive learning** to make exam preparation fun, effective, and efficient.

## 🎯 How It Works

### For Students:

1. **Sign Up & Login** - Create your account securely
2. **Create Custom Quizzes** - Paste any study material (notes, articles, textbooks) and let AI generate relevant MCQs
3. **Practice Quizzes** - Take interactive quizzes with immediate feedback
4. **Track Progress** - Monitor your performance with real-time statistics
5. **Retake Quizzes** - Practice multiple times and see your improvement

### Behind the Scenes:

- Uses **Gorq AI** to automatically generate multiple-choice questions from your study material
- Stores all data securely in **Supabase** database
- Provides instant scoring and performance analytics

## 💡 Key Features

✅ **AI-Powered Question Generation** - Converts any text into relevant MCQs automatically
